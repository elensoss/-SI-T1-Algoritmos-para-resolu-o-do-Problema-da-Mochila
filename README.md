
Este repositório contém a implementação de um **Algoritmo Genético (AG)** para resolver o Problema da Mochila 0/1.

## Oráculo 
Para a comparação dos resultados, utilizamos o algoritmo **MINKNAP**, desenvolvido pelo pesquisador **David Pisinger**. 

* **Arquivo:** `oraculo.cpp` (Implementação original de Pisinger em C).
* **Função:** Este algoritmo resolve o problema de forma exata usando programação dinâmica otimizada, servindo como a comparação para os resultados obtidos pelos algoritmos que implementamos.
- **Site Oficial:** [David Pisinger's Optimization Codes](http://www.diku.dk/~pisinger/codes.html)
